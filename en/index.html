Welcome to TraceFiction

TraceFiction is a page dedicated to reading fiction generated software.

iPhone / iPad users to read pages often encountered with the following prombles: 
How to continue reading next time?
Scrolling is difficult to page down/ page up?
How to track new chapters? 
Font and background does not look good
Unable to full-page view.
Read Chinese Simple/Traditional words. 
TTS.


In response to these problems, the TraceFiction was created , TraceFiction has the following features: 

1 record your last viewing position. 
2 specific site has Javascript re-layout format, adjust the font size and color.
3 side swipe page up/page down. 
4 support TonWenTang character conversion. 
5 support tracking mode and normal mode. 
6 supports multiple iOS 7 TTS function, read footer from the left corner, select the text to read, select the text to read until footer. 
7 support wire, Bluetooth headsets and other playback controls 
8 support background read mode


First, the page is divided into two TraceFiction modes - normal mode and tracking mode.
Under normal mode, TraceFiction similar general web browser , the different are that it only provides "desktop mode" full page view. Previous, Next and navigate all shall gesture control operations.

Normal mode provides functions: Previous, Next, reload the page, turn one on the next turn one, TTS (text to speech), zoom and fit the screen functions.

Tracking mode is different to general web browsing quite , it will be synchronized to update records where are you reading, that is, tracking list will record your last reading position, the next time you turn on automatically jump to the last location, and can simultaneously track multiple book.
After the page loads, it will automatically translate traditional or simplified chinese with the setting. Certain sites, support Javascript automatic layout, page will become more suitable for reading novels. Formats such as exhibited on the App Store.

Tracking mode provides functions: Previous, Next, reload the page, turn one on the next turn one, TTS (text to speech), character conversion, Javascript format conversion.
There are four parts in the TraceFiction, the first part is the browser, and the second part is the navigation, and the third part is the rest reminder and advertising, and the fourth part is the gesture instructions.

The first part of the control method of the browser as follows: 

2 taps of middle part on the screen: navigation page 
2 taps of left part on the screen: Previous 
2 taps of right part on the screen: forward
3 taps: start or pause TTS
A finger swipe: page scroll 
Swipe from left edge : page up (can be customized) 
Swipe from right edge page down (can be customized) 
long press: select content, finished selecting options appear TTS


The navigation page control methods are as follows:

Navigation page is divided into navigation, text and reading list 

<: Previous 
>: Next Page 
+: Add to Wish List 
X: Close button 
The address bar: You can enter the URL or keyword search by Google. 
Go: Start a new reading or searching. 
Font settings: You can set the text color, background and size, have set up to reload the page (tracking mode only) 

Track Listing: 
  Left slide: there may be deleted 
  Double-tap: Set to normal mode or tracking mode 
  Tap: Start Reading


The third part of the rest reminder ad pages and control methods are as follows: 

Each hour will remind need to let the eye rest. 
Close press advertising, or the upper right corner of the Close button. Once every three hours basis having advertising, sponsorship about the author (^_^).


The fourth part of the page is a gesture control method described as follows: 

It can set don't display again. ( user can reset it in the setting).
